HTTP Status Codes

HTTP Status Code ครบทุกตัว • ภาษาไทย • Ultra UI Animation

1xx • Informational

สถานะระหว่างการประมวลผลของเซิร์ฟเวอร์

  • 100 Continue – ส่งข้อมูลต่อได้
  • 101 Switching Protocols
  • 102 Processing
  • 103 Early Hints
2xx • Success

คำขอถูกต้องและสำเร็จ

  • 200 OK
  • 201 Created
  • 202 Accepted
  • 203 Non-Authoritative
  • 204 No Content
  • 205 Reset Content
  • 206 Partial Content
  • 207 Multi-Status
  • 208 Already Reported
  • 226 IM Used
3xx • Redirection

ต้องเปลี่ยนเส้นทางไปยัง URL อื่น

  • 300 Multiple Choices
  • 301 Moved Permanently
  • 302 Found
  • 303 See Other
  • 304 Not Modified
  • 307 Temporary Redirect
  • 308 Permanent Redirect
4xx • Client Error

ข้อผิดพลาดจากฝั่งผู้ใช้

  • 400 Bad Request
  • 401 Unauthorized
  • 403 Forbidden
  • 404 Not Found
  • 409 Conflict
  • 422 Unprocessable Entity
  • 429 Too Many Requests
  • 451 Legal Reasons
5xx • Server Error

ข้อผิดพลาดจากฝั่งเซิร์ฟเวอร์

  • 500 Internal Server Error
  • 501 Not Implemented
  • 502 Bad Gateway
  • 503 Service Unavailable
  • 504 Gateway Timeout
  • 511 Network Authentication Required
